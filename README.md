# 🤖 ULTRABOT - Robô de Trading Inteligente

Robô de trading automatizado para Bybit com foco em lucros diários.

## 🚀 Características

- 🤖 **Inteligência Artificial**: Múltiplas estratégias combinadas
- 💰 **Foco em Lucros**: Meta diária de 5%+
- ⚡ **Modos Operacionais**: Agressivo e Conservador
- 🛡️ **Sistema de Defesa**: Proteção contra losses
- 📱 **Telegram Integration**: Controle total via bot
- 🌐 **Web Dashboard**: Interface profissional
- 🔄 **Aprendizado Contínuo**: Melhora com o tempo

## 🛠️ Instalação

1. Clone o repositório
2. Configure as variáveis de ambiente no `.env`
3. Instale as dependências: `pip install -r requirements.txt`
4. Execute: `python app.py`

## 📊 Comandos Telegram

- `/start` - Iniciar bot
- `/status` - Status do trading  
- `/iniciar` - Começar operações
- `/parar` - Parar trading
- `/modo agressivo|conservador` - Mudar modo
- `/lucros` - Ver lucros
- `/estatisticas` - Estatísticas

## ⚙️ Configuração Bybit

1. Crie API Key na Bybit
2. Configure permissions: Trade, Wallet Read
3. Adicione no `.env`:
# ULTRABOT
